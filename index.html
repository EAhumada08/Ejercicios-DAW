JavaScript

JavaScript es un lenguaje de programación interpretado, dinámico y orientado a objetos que se utiliza principalmente para agregar interactividad y dinamismo a las páginas web. Aunque inicialmente fue diseñado para la web, hoy en día se usa también en aplicaciones del lado del servidor (como en Node.js) y en otros entornos no relacionados con el navegador.
